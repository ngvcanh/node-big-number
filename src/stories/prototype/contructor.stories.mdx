# BigNumber.prototype.constructor()

```tsx


BigNumber.prototype.constructor(value?: BigNumberData): BigNumber;
```

Initialize a big number. The `value` parameter is defaut value of object. 
If the `value` parameter is not passed, default value is empty string.

## Usage

```tsx
import BigNumber from '@kensoni/big-number';

const bigA = new BigNumber('11111111111111111111111111111111111111111111.33333333');
console.log(bigA);

/** Output:
  BigNumber {
    _value: "11111111111111111111111111111111111111111111.33333333" 
    _integer: "11111111111111111111111111111111111111111111" 
    _decimal: "33333333"
    _nevigate: false 
  }
*/

const bigB = new BigNumber('-11111111111111111111111111111111111111111111.33333333');
console.log(bigB);

/** Output:
  BigNumber {
    _value: "-11111111111111111111111111111111111111111111.33333333" 
    _integer: "11111111111111111111111111111111111111111111" 
    _decimal: "33333333"
    _nevigate: true 
  }
*/
```

In addition, method [BigNumber.from()](./?path=/story/static-bignumber-from--page) can be used to initialize object `BigNumber`.
